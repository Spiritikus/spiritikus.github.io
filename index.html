📄➡️📚 Конвертер тестов: Word → Moodle XML

Преобразуйте текст вопросов (из Word, текстовых файлов) в формат импорта Moodle XML (вопросы с выбором одного ответа)
📝 Исходный текст (Word / TXT)
📌 Правила разметки (один из способов):
• Каждый вопрос начинается с цифры и точки/скобки (1., 2. и т.д.)
• Варианты ответов: A) Текст / B. Текст (буквы + разделитель)
• Указание правильного ответа: Правильный ответ: B или Answer: C
• Вопросы отделяйте пустыми строками. Поддерживается копирование из MS Word.
🎓 Moodle XML (готовый к импорту)
📥 Импорт в Moodle: Созданный XML совместим c категорией вопросов Moodle (тип "выбор одного ответа"). Используйте Администрирование → Банк вопросов → Импорт → Формат Moodle XML.
🔍 Поддержка множественного выбора из текста: автоматический поиск вариантов A), B), C), D) или A., B. и т.д. Правильный ответ определяется по ключевой фразе. Работает с русским/английским текстом, извлечение из .docx (Word) осуществляется встроенным парсингом (библиотека mammoth не требуется — упрощённый парсинг через FileReader + чистка). Для корректной обработки сложных .docx используйте предварительное копирование текста.
